# Конфигурация

Noctis подключается к серверу через адаптер. Тип адаптера задаётся в `AdapterConfig`.

## Выбор адаптера

В `lib/core/di/providers.dart` есть `adapterConfigProvider`:

```dart
final adapterConfigProvider = StateProvider<AdapterConfig>((ref) {
  return const GravitConfig(host: 'localhost', port: 9274);
});
```

Замени на нужный тип:

<Tabs items={['GravitLauncher', 'Custom REST', 'Mock']}>
  <Tab value="GravitLauncher">

```dart
const GravitConfig(
  host: 'mc.example.com',
  port: 9274,
  useTls: false,
)
```

Протокол WebSocket. Подробности в `docs/gravit_protocol.md`.

  </Tab>
  <Tab value="Custom REST">

```dart
const CustomRestConfig(
  baseUrl: 'https://api.example.com',
)
```

HTTP-эндпоинты через Dio. Токены обновляются автоматически через interceptor.

  </Tab>
  <Tab value="Mock">

```dart
const MockConfig()
```

Хардкоженные данные, никуда не подключается. Удобно для вёрстки и тестов UI.

  </Tab>
</Tabs>

## Java

Путь к Java настраивается в экране Settings внутри лаунчера. По умолчанию используется `java` из PATH. Для Minecraft нужна Java 17+.

## Память

RAM для игры тоже задаётся в Settings. Дефолт: min 512MB, max 2048MB.
