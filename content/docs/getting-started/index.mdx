# Noctis

Noctis — десктопный лаунчер для Minecraft. Подключается к серверам через адаптеры (Gravit, REST API или мок для разработки), проверяет целостность файлов, скачивает обновления и запускает игру.

## Что умеет

- **Мультибэкенд** — работает с GravitLauncher (WebSocket), кастомным REST API или моком. Переключается одной настройкой.
- **Автообновление клиента** — синхронизация файлов с сервера, SHA-1 проверка после скачивания.
- **Система тем** — UI полностью заменяемый. Из коробки идёт Void Default с glassmorphism и тёмной палитрой.
- **Защита от читов** — pre-launch проверка файлов, детект DLL-инжекта через мониторинг модулей процесса, наблюдение за файлами во время игры.
- **Мультиплатформенность** — Windows (основная), Linux, macOS.

## Стек

| Технология | Для чего |
|-----------|----------|
| Flutter / Dart | UI и логика |
| Riverpod | Стейт-менеджмент и DI |
| GoRouter | Навигация |
| Freezed | Иммутабельные модели |
| Dio | HTTP-клиент |
| dart:ffi | Нативные вызовы Windows API |

## Архитектура

Noctis построен на Clean Architecture:

```
domain/     — бизнес-логика, сущности, интерфейсы репозиториев
data/       — реализация репозиториев, работа с хранилищем
core/       — адаптеры, DI, конфиг, ошибки, security, темы
```

Зависимости идут только внутрь: `presentation → domain ← data`. Domain не знает ни про Flutter, ни про сеть.
