# Установка

## Требования

- Flutter SDK 3.41+
- Dart SDK 3.11+
- Git
- Для Windows: Visual Studio с C++ workload (для Flutter desktop)

<Steps>
  <Step>

### Клонирование

```bash
git clone https://github.com/Noctralius/Noctis.git
cd Noctis
```

  </Step>

  <Step>

### Зависимости

```bash
flutter pub get
```

  </Step>

  <Step>

### Кодогенерация

Модели используют Freezed, поэтому после клонирования нужно запустить build_runner:

```bash
dart run build_runner build --delete-conflicting-outputs
```

  </Step>

  <Step>

### Запуск

<Tabs items={['Windows', 'Linux', 'macOS']}>
  <Tab value="Windows">
```bash
flutter run -d windows
```
  </Tab>
  <Tab value="Linux">
```bash
flutter run -d linux
```
  </Tab>
  <Tab value="macOS">
```bash
flutter run -d macos
```
  </Tab>
</Tabs>

  </Step>
</Steps>

## Сборка релиза

<Tabs items={['Windows', 'Linux', 'macOS']}>
  <Tab value="Windows">
```bash
flutter build windows --release
```
Бинарник: `build/windows/x64/runner/Release/`
  </Tab>
  <Tab value="Linux">
```bash
flutter build linux --release
```
Бинарник: `build/linux/x64/release/bundle/`
  </Tab>
  <Tab value="macOS">
```bash
flutter build macos --release
```
  </Tab>
</Tabs>
